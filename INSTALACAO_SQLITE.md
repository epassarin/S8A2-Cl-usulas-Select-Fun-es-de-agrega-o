# ✅ Plugins/extensões do SQLite para VS Code

## 🔴 OBRIGATÓRIO (instalação)

Você **precisa de pelo menos um destes** para trabalhar com SQLite no VS Code.

### 1️⃣ **SQLite**

**Autor:** alexcvzz  
✅ **Mais usado e recomendado**

**Funções:**

*   Abrir arquivos `.db`, `.sqlite`, `.sqlite3`
*   Executar comandos SQL
*   Visualizar tabelas e dados
*   Interface simples e integrada

👉 **Para a maioria dos projetos, só ele já resolve**

***

## 🟡 RECOMENDADOS (complementam o uso)

### 2️⃣ **SQLite Viewer**

**Autor:** Florian Klampfer

**Funções:**

*   Visualizar bancos SQLite
*   Somente leitura (mais segurança)

👉 Ideal para **consultar dados rapidamente** sem risco de alterar

***

### 3️⃣ **SQLTools**

**Autor:** mtxr

⚠️ Sozinho **não funciona** para SQLite — precisa do driver abaixo.

**Funções:**

*   Gerenciar conexões
*   Autocomplete avançado
*   Organização profissional de bancos

***

### 4️⃣ **SQLTools SQLite**

**Autor:** mtxr

✅ **Driver do SQLite para o SQLTools**

**Funções:**

*   Permite que o SQLTools se conecte ao SQLite

👉 **Sempre instalar junto com o SQLTools**

***

## 🟢 OPCIONAIS (melhoram produtividade)

### 5️⃣ **SQL Formatter**

**Funções:**

*   Formatar código SQL automaticamente
*   Padronizar queries

***

### 6️⃣ **Database Client**

**Funções:**

*   Cliente gráfico de banco de dados
*   Interface tipo DBeaver
*   Suporte a SQLite e outros bancos

***
